Lógica de Programação com Visualg :
Este repositório contém exercícios e práticas desenvolvidas em Visualg, utilizando pseudocódigo em português estruturado.
O objetivo é fortalecer a base em lógica de programação, estruturas condicionais, estruturas de repetição e manipulação de variáveis.

Conteúdo :
* Estruturas condicionais (SE / SENÃO)
* Estruturas de repetição (PARA, ENQUANTO)
* Operadores lógicos e matemáticos
* Exercícios de fixação
* Problemas práticos resolvidos em pseudocódigo

Objetivo:
Desenvolver raciocínio lógico e preparar a base para linguagens como Java, C, Python e outras.

Ferramenta utilizada:
Visualg
